CASIO fx-991ES PLUS — Complete Classroom Guide

All major functions with specific mode keys and short “how to do” steps, based on your attached instruction set.

1. Basic Calculations

Calculator reference image
Keypad layout
Display guide

Arithmetic (+, −, ×, ÷)

  1. Enter expression (example: 5 + 3 × 2).
  2. Press =.

Fractions

  1. Press a b/c.
  2. Enter numerator, then denominator.
  3. Press =.

Convert: SHIFT + a b/c to switch fraction ↔ decimal.

Powers & Roots

  • Power: x² or ^
  • Root: SHIFT + x² for √
√25 → SHIFT + x² → 25 → =

2. Scientific Functions

Trig keys
Mode keys
Example display

Trigonometry (sin, cos, tan)

  1. Set angle mode: MODE → 3 (DEG) or 4 (RAD).
  2. Enter function, angle, then =.
sin(30) → sin → 30 → =

Inverse Trig

SHIFT + sin + value + =

Logarithms

  • log (base 10)
  • ln (natural log)
log(100) → log → 100 → =

3. Algebra & Equation Solving

Solve Polynomial Equations

  1. MODE → 5 (EQN)
  2. Choose 2 (quadratic) or 3 (cubic).
  3. Enter coefficients.
  4. Press = repeatedly to view solutions.

Solve Simultaneous Equations

  1. MODE → 5
  2. Choose number of unknowns (2 or 3).
  3. Enter coefficients row by row.
  4. Press = to cycle solutions.

4. Table Function (Function Evaluation)

Generate Table of Values

  1. MODE → 7 (TABLE)
  2. Enter function (example: x²).
  3. Press =.
  4. Enter start, end, step.
  5. Scroll through results.

5. Calculus Features

Numerical Derivative

  1. SHIFT → d/dx
  2. Enter function.
  3. Enter evaluation value.
  4. Press =.

Definite Integral

  1. SHIFT → ∫
  2. Enter lower limit, upper limit, function.
  3. Press =.

6. Complex Numbers

Complex Mode

  1. MODE → 2 (CMPLX)
  2. Use SHIFT + . for i.
(2 + 3i) + (1 + i)

7. Matrices

Matrix Input

  1. MODE → 6 (MATRIX)
  2. SHIFT → 4 (Matrix menu)
  3. Select 1: Dim.
  4. Choose matrix size.
  5. Enter values.

Matrix Operations

  • Determinant: SHIFT → 4 → 7
  • Inverse: x⁻¹

8. Vectors

Vector Mode

  1. MODE → 8 (VECTOR)
  2. Define vectors via SHIFT → 5.
  3. Perform dot product and magnitude operations.

9. Statistics

Statistical Calculations

  1. MODE → 3 (STAT)
  2. Choose type (1-variable or regression).
  3. Enter data.
  4. SHIFT → 1 for STAT menu.

Get: Mean, Standard deviation, Regression equations.

10. Base-N Calculations

Binary / Hex / Decimal

  1. MODE → 4 (BASE-N)
  2. Select base (BIN, OCT, DEC, HEX).
  3. Enter number.
  4. Convert by pressing another base key.

11. Constants & Conversions

Physical Constants

  1. SHIFT → 7 (CONST)
  2. Select constant.
  3. Press =.

Unit Conversion

  1. SHIFT → 8 (CONV)
  2. Choose conversion type.
  3. Enter value and press =.

12. Memory Functions

Store Value

number → SHIFT + STO → variable (A–F, X, Y, M)

Recall

RCL → variable

13. Replay & Editing

Edit Previous Expression

  • Use arrow keys ↑ ↓ ← →.

14. Setup & Display Modes

Change Display Format

  1. SHIFT → MODE (SETUP)
  2. Choose MathIO/LineIO or Fix/Sci/Norm.

15. Random & Misc

Random Number

SHIFT + . (Ran#)

16. Factorial, nCr, nPr

Factorial

number → SHIFT + x! → =

Combinations / Permutations

n → SHIFT + nCr → r n → SHIFT + nPr → r

17. Angle Conversions

Convert DEG ↔ RAD

SHIFT + DRG
